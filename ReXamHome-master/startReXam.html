OPTW-Easy proctoring,Right from your home
Screen+Camera Recording for Exam from Home
Instructions:
  1. Course ID
  2. Student ID
  3. Start Rec.
  4. Stop Rec.
  5. Choose File Checksum
  6. Cal. Checksum: Calculates Checksum values
  7. Send Checksum + video files (eg via SMS, Checksum only, Online Examination System, Google Form)
Options:
  • Monitor: Choose to enable or disable the screen showing recorded videos.
  • Audio Rec.: Select to record sound.
  • Course ID/Student ID: It will be used as the name of the recorded video file. If not filled out Will use the date and time of the exam as the file name instead
